Example of a function that is once but not differentiable

Posted on Oct 17, 2016 in Examples

A discontinuous function, such as that of $h(x) = |x|$, is an example of a non-differentiable function. How about a function that is once differentiable but not twice? Well, integrate $h(x)$ and define the resulting function $f$ at 0 so that the result is continuous.

$$ f(x) = \begin{cases} -\frac{1}{2} x^2 & \text{if $x < 0$}\\ \phantom{-}\frac{1}{2} x^2 & \text{otherwise} \end{cases} $$

def f(x):
    if x < 0:
        return -0.5 * x**2
    else:
        return 0.5 * x**2
f = np.vectorize(f)
